GIMP Practice 1

 

Practice 1:  Working with Images in GIMP

Refer to Chapter 1 & 2 in your textbook, Beginning GIMP: From Novice to Professional.

 

Step 1.    Download the GIMP1 folder found in the GIMPPractice1 folder in WebCT. 

 

Step 2.    Opening different file formats.

               Open the moon.jpg file.  This file is opened as a bitmapped or Raster image.

 

               Open the ForeverInLove.png image.

               This file is opened the text on a transparent background. 

              

               Open the Mummy.gif image.

Look in the Filters=>Animation=>Playback.  Click on Play/stop or Step to see the animation.

               Notice the animated GIF is made up of ten frames.

               We will learn how to create these animations in a later lesson.

 

Step 3.    Making Canvas modifications:

               Open a new file in GIMP.        

               Set the Canvas Size to 400px x 400px.

               Choose to set the background.

 

Step 4.    Cropping Images:

               Open the composite.jpg file. 

               Choose the crop tool from the Tool Bar.

Hold down the left mouse button and draw  a rectangle around the starship enterprise.

               Click on the crop button to crop the image. 

 

Step 6.    Transforming Images:

               Open the Space Shuttle image.

               Choose  Transform and Rotate 90 BCW from the Modify Menu.

               Choose  Transform and Rotate 90 BCCW from the Modify Menu.

               Choose Transform and Distort from the Modify Menu. 

               Distort the image by dragging on one of the handles.

               Choose Transform and Flip Vertical from the Modify Menu.

 

Step 7.    Adding Text to an Image:

               Select the Text Tool from the Tool Bar.

               Drag a rectangular area and type text.

               Enter the text: Space Shuttle Columbia. 

               Set the font, font size and font color using the Properties Inspector.

               Click Deselect the text.

 

Step 8.    Saving your file as a XCF:

               Save your file as the default GIMP XCF.

               Always do this so you can go back and modify your work later.

 

 

Step 9.    Exporting Images as a JPEG:

               Choose the Export Wizard from the File Menu. 

               Select Continue and then select the Web as the Destination. 

               Change the compression rate of the JPEG image so the file size is smaller.

               Save the image as Shuttle.jpg.

 

Conclusion:  Whether you are creating new images, or importing images to be modified, GIMP is a powerful tool for Web development.  Later we will learn how to create text graphics, an imagemap and animations for the web.
