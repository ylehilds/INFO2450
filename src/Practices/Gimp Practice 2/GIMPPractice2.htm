Using Layers, Effects and Text in GIMP

Step 1.  How to create a new layer.
               Set the background color to be a light blue. 

               Make sure blue it is the color behind in the color area of the toolbar.

               Open a new file in GIMP.

               Set the Canvas size to have 400 pixels Width and 300 pixels Height.
               Click on the Advanced Options in the Create a New Image Dialog box.

Set the Fill With color to be the background. Click the OK button.
If the Layers Panel is not already opened, use the Dialog=>Layers menu option to open it.

               Notice you now have one layer named background in your image.
               Click on the layer name once to select it and rename the layer to: Blue Skies.         

               Save the file as SnoopyFinal.xcf in the GIMP1 Images folder.

 

Step 2.  Open existing image as a layer.

Choose the File=>Open as Layer option. 
Browse to find the Snoopy2.gif in the Images folder of this practice folder and click Open.
You can click on the Move Layers & Selections tool in the GIMP toolbar and click on Snoopy to move him to the right.
Open another image as a layer.
Choose the File=>Open as Layer option.
Browse to find the Woodstock.gif file and click Open.

 

Step 3.  Add Cloud effects to the Background.

Select the Background layer.
Choose Filters=>Render=>Clouds=>Solid Noise.
Set the Random Seed and Detail values and adjust the X and Y size until you have a cloud pattern you want, then click on OK.
To get a blue and white coloration in the clouds choose Layers=>Colors=>Colorize. 
Adjust the Hue to the right until you have a blue color.
Now adjust the Brightness to the right until you have the light blue you want then click OK.
If you want you can also adjust the Saturation a bit to the right.

 

Step 4.  Add Cloud effects to the Background.

Select the Background layer.
Choose Filters=>Render=>Clouds=>Solid Noise.
Set the Random Seed and Detail values and adjust the X and Y size until you have a cloud pattern you want then click on OK.

 

Step 5.  Scale Woodstock much smaller.

               Click on the layer with Woodstock in it.

Notice there is a selection rectangle now around Woodstock.
Choose the Scale Layer or Selection from the GIMP toolbar and click on Woodstock.
This will bring up the Scale Dialog box.  If it is in your way, you can drag it to the side by clicking
in the blue title area and dragging it.
Decrease the current width to about 25 px.  Notice the height scales proportionately.  Then click on Scale to complete the scaling operation.
Now click on the Move Layers & Selections tool in the GIMP toolbar and move Woodstock to sit on Snoopys scarf.

Step 6.  Adding Text as a Layer.

Click on the Add Text to Image tool in the GIMP toolbox. 
Now click your cursor on the image.
The Gimp Text Editor dialog box should appear.
Type the following: Snoopy and the Red Baron.  Put returns where you want.
In the Text Options area of the toolbox, set the Font to be Comic Sans MS.
Set the Size to be 25.
Set the Color to be Red.
And set the Justify to be centered.
Click on the Text Layer.
Choose  the Move Layers & Selection tool from the GIMP toolbox.
Select the Text in the image with the finger of the hand icon.
Use the right arrow key to move the text to the left side of Snoopy.

 

Step 7.  Adding a Drop Shadow Effect to the Text.

               Click on the Text layer.
               Choose the duplicate layer icon at the bottom center of the Layers Panel.
               Click on the eye (Visibility icon) of the top Snoopy Text layer to hide it.

Click on the second Snoopy text layer and rename it: Shadow.
With the Shadow Text layer selected.  Use the Text tool and click on the Shadow text.
Go to the Text options panel of the GIMP toolbox and change the color from red to black.
Change the Font to be Comic Sans MS Bold.
Then select the Filter=>Blur=>Gausian Blur menu option to blur the shadow text.

Click the Preview to see the shadow effects. 
Modify the Horizontal and Vertical Blur Radius until you achieve the shadow effect you want.
When you are done, click on OK.

 

Step 8.  Final Text Placement and Cropping of Image.

In the Layers Panel, click on the eye of the Snoopy Text layer so it is visible again.
Use the arrow keys to move the red Snoopy text where you want it over the Shadow layer.
Now in the Layer Panel, click on the chain to the left of the Snoopy Text icon and the chain to the left of the Shadow Text icon.  This will now chain the two Text layers together.
Using the arrow keys, move the Snoopy and Shadow text over the top of Snoopys doghouse.
The word Snoopy should be at the top of the dog house.

We will now save the image in its XCF form again so we preserve the layers if we ever want to come back and modify the image.
Now making sure all the layers are visible, right click on the Layers Panel and choose Merge Visible Layers.  Then click OK.
You just have the one Background layer now.
Use the Cropping tool from the GIMP toolbox. Holding down the left mouse button, drag a rectangle around the image where you want to crop it.  If the Cropping dialog box is in your way.

Hold down both the Shift key and the left mouse button when you drag the cropping rectangle.
Click on the Crop button in the Cropping Dialog box to complete the cropping.

 

Step 9.  Now we will export this final image as a JPEG file. 
Choose Save as from the File menu.
Save the image as SnoopyAndWoodstock.jpg.  Choose the file extension as JPEG.
If you are prompted to Flatten the image to remove transparency, choose Export.
Set the JPEG compression to 80 and click OK.
You now have a cool JPEG image you have created.

Conclusion:  Layers are the most useful feature of GIMP and many other graphical applications use this feature also.  Text can be enhanced by adding effects and styles.  Remember to always save your GIMP files as their native format, XCF.  This will allow you to go back and modify the file if necessary.  If exporting your file for the web, save your graphics as either a GIF or JPEG file.  If using transparency, you must export the file as a GIF.  Have fun creating your own graphics.
