After you prepare the system, you can install the board:

  1. Slide the board into place while guiding the interface through the slot in the back of the chassis.
  2. Mate the pins on the board with the socket on the chassis wall.
  3. Press down firmly to seat the board.
  4. Install the anchoring screws to prevent the board from coming loose.

NOTE: Do not install the board with the plastic wrapping still covering it. Your board may not operate properly under these conditions.
