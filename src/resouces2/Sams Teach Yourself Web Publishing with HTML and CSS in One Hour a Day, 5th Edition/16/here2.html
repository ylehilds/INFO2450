find-foo is a tool for finding instances of the word "foo" in source code files. Several versions are available:

  • Version 2.4 is the most recent released version. It is available for ftp at ftp.find-foo.com/pub/find-foo. Documentation for release 2.4 is also available.
  • Previous versions of find-foo, with binaries and documentation, are archived at ftp.find-foo.com/pub/find-foo/old.
  • The most recent unsupported beta is also available at ftp.find-foo.com/pub/find-foo/beta.
