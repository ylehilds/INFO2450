Adding Links: HTML & CSS Lesson 5

The <a> anchor tag is used for creating links.  There is an opening and a closing anchor tag, <a> and </a>

A link needs two things:  a filename and some hypertext.

Know how to use relative and absolute file pathnames.

Use relative pathnames for pages in your site and absolute pathnames for non-local pages.

Filenames and pathnames must be spelled exactly or they usually will not work!!!

Use the href attribute to create a link to a file.

            <a href= “circle.html”> Circles </a>

Hypertext appears between the open and closing anchor tags.

Use the name attribute to create an internal link destination.  Provide hypertext for the link destination. 

            <a name= “top”>My Favorite Things </a>

Use the href and the # to jump to an internal link:

            <a href= “top”> Return to Top</a>

To jump to an internal link on another page use the filename and the #name.

            <a href= “circle.html#top”>My Favorite Things </a>

Know the different types of URLs.

Adding Images: HTML & CSS Lesson 7

Know the two types of graphics: vector and raster.

Web pages usually use raster graphics.

Web color is made up of RBG- Red, Blue and Green

24 bit true color is made up of - 8bits Red, 8bits Blue and 8bits Green

You can display millions of colors using 24 bit color

Colors can have a name or a Hexidecimal number value

There are only 216 Web-safe colors

Tables – HTML & CSS Lesson 8

Know the four uses of tables.

Creating a Data table is covered in this lesson.

The basis structure of tables is rows and columns.

Know these table tags:

            <table> , </table>

            <tr> , </tr>

            <th> , </th>

            <td> , </td>

Know how to label your table with a caption.

Know the attributes of <caption>

Know these table level controls:

            width ( number or percentage - width of entire table)

            align (left, center, right)

            bgcolor (hexadecimal value or name of color- background color or entire table)

            border (number of pixels thick)

            cellpadding (distance from cell contents to edges of cell)

            cellspacing ( the amount of space between cells)

Know these row level controls:

            align (left, right, center - horizontal alignment of entire row content)

            valign ( top, middle, bottom, baseline- vertical alignment of entire row contents)

            bgcolor ( hexadecimal value or name of color - background color of entire row)

Know these cell level controls:

            align (left, right, center - horizontal alignment of cell content)

            valign ( top, middle, bottom, baseline- vertical alignment of cell contents)

            bgcolor ( hexadecimal value or name of color - background color of cell)

            width( number or percentage- width of individual cell)

            height ( number or percentage - height of a cell, entire row has max height of cell)

            colspan ( number- number of columns a cell spans)

            rowspan (number - number of rows a cell spans)

Know the priority of level of controls of a table:

            Cell attributes have highest priority

            Then row attributes

            Then table attributes

            Finally attributes set outside of table have lowest priority

 

Know these advanced table topics:

            Grouping and aligning columns - <colgroup> , </colgroup>

                        span - number - number of columns

                        width - number or percentage- width of column

                        Use <col> to vary the attributes of columns in the group

            Grouping and aligning rows

                        <thead> - sets apart a head of a table

                        <tbody> - sets apart the body of a table

                        <tfoot> - sets apart the footer of the table

            Frame attributes - set how borders or frames of table can look

            Rule attributes - draws rules within the table

Text Alignment Tables:

            Used to align text

            Usually sets border to 0

            Empty cells can be used for indentation

Page Template Tables:

            Used to layout page content

            Two common layouts: horizontal and vertical

Multipart Image Container Tables:

            Used to display image slices so page loads quicker

            Border, cellpadding and cellspacing should all be set to 0

            Make sure there are no extra spaces between <img> tag and <td> tags

Tables within Tables:

            Inner table must be placed within a <td> tag!!!

            Allows creation of more complex table structures
